Project Overview

Developed an interactive Street Fighter 6 Power BI dashboard to analyze player performance, match outcomes, and character usage trends. The dashboard consolidates gameplay and match-level data to provide clear insights into win rates, play styles, and competitive balance.

Key Features Implemented:


▸Integrated multiple datasets including match results, character statistics, and player rankings

▸Built 12+ interactive Power BI visuals with slicers for character, rank, and match type

▸Designed drill-through pages for detailed player and character performance analysis

▸Implemented calculated measures using DAX for win rate, pick rate, and performance trends

▸Created an executive-style summary page highlighting top insights and trends

Challenges & Solutions

⚡
Challenge

Street Fighter 6 match data existed in raw, unstructured formats, making it difficult to identify performance trends, character balance, and player progression. Manual analysis was time-consuming and lacked interactivity.

✦Solution

Developed a centralized Power BI dashboard with cleaned and modeled datasets. Applied data transformations in Power Query, created optimized DAX measures, and designed an intuitive layout allowing users to filter, drill down, and explore performance insights dynamically.

Results & Outcomes

Reduced manual analysis time by over 80% through automated dashboards
Enabled fast comparison of player and character performance across ranks
Identified top-performing characters and matchup trends
Improved strategic decision-making for competitive analysis and gameplay optimization
